Source Example
==============
lab2round1orc provides functions for describing an orc

Determining Orc Type
^^^^^^^^^^^^^^^^^^^^

The function :func:`source.lab2round1orc.get_orc_type` provides users with a way to get
the type of orc (types are given by number)

Type Example
^^^^^^^^^^^^^^^

>>> from source.lab2round1orc import get_orc_type
>>> orc = Orc(100, 50, 2)
>>> get_orc_type()
'2'
