Source Example
==============

lab2defense provides functions for implementing defense.

Detect Breach
^^^^^^^^^^^^^

The function :func:`source.lab2defense.detect_breached` provides users with a way to see if 
their defenses have been breached, and returns (True, or False)

defense Example
^^^^^^^^^^^^^^^

>>> from source.detect_breached import detect_breached
>>> detect_breached(90, 100)
'True'
